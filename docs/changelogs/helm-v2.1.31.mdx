## Changelog

- Added `bifrost.guardrails.rules[].stream_replay_event_interval_ms` (default `0`, range 0-1000) to configure the delay in milliseconds between consecutive buffered events after block-capable streaming output guardrails allow the response; `0` sends the buffered events immediately.
