This note documents the clean branch used for Bedrock fine-grained tool streaming work based on the `v1.5.0-prerelease3` transport release line.

## Goal

Use a branch that stays as close as possible to the upstream `maximhq/bifrost:v1.5.0-prerelease3` image behavior, while carrying only the changes required for:

- Bedrock passthrough `extra_params`
- Anthropic-compatible `eager_input_streaming`
- Fine-grained tool streaming beta propagation
- 600-second stream idle timeout
- Fork-friendly GHCR image publishing

This branch intentionally avoids unrelated schema drift and migration fixes that were added later on a different experimental branch.

## What Changed

### Runtime behavior

- Preserves Anthropic-compatible `extra_params` through the Anthropic -> Bifrost -> Bedrock conversion path.
- Adds `eager_input_streaming` support on Anthropic-compatible tools.
- Preserves the flag into Bedrock tool specs.
- Treats `fine-grained-tool-streaming-*` as an allowed Bedrock Anthropic beta.
- Increases default stream idle timeout from 60s to 600s.
- Disables Bedrock streaming client's total `http.Client.Timeout` so long-running streams are governed by idle timeout instead of a hard request timeout.

### CI and image publishing

- Adds a fork-friendly GitHub Actions workflow at `.github/workflows/ghcr-image.yml`.
- Publishes to `ghcr.io/<repo_owner>/bifrost`.
- Defaults to `linux/amd64` builds for faster and more predictable fork publishing.

## Build a Fork Image

### GitHub Actions

Pushing a tag that starts with `v` triggers the GHCR workflow:

```bash
git tag -a v1.5.0-prerelease3-bedrock-extra-parameters-clean -m "Build clean prerelease3 image"
git push tokenroolai refs/tags/v1.5.0-prerelease3-bedrock-extra-parameters-clean
```

Expected image:

```bash
ghcr.io/tokenrollai/bifrost:v1.5.0-prerelease3-bedrock-extra-parameters-clean
```

### Manual local build

Use the local-module Dockerfile so the build includes local `core/`, `framework/`, `plugins/`, and `transports/` sources:

```bash
docker build \
  -f transports/Dockerfile.local \
  --build-arg VERSION=1.5.0-prerelease3-bedrock-extra-parameters-clean \
  -t ghcr.io/tokenrollai/bifrost:v1.5.0-prerelease3-bedrock-extra-parameters-clean \
  .
```
